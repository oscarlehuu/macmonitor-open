# MacMonitor

A lightweight, native macOS menu bar app for monitoring system resources on Apple Silicon Macs.

## Features

- **Memory** — Real-time RAM usage, pressure breakdown, and per-process memory tracking
- **Storage & Apps** — Disk usage analysis, cleanup recommendations with safety labels, and app uninstall
- **Network** — Live upload/download rates in the menu bar
- **Battery** — Health percentage, cycle count, charging policy management
- **Trends** — 24-hour and 7-day usage graphs for memory and storage
- **Widget** — Desktop widget for at-a-glance system stats

## Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon (M1/M2/M3/M4)
